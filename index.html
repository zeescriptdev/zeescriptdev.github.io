Questions, answered

Which AI subscription do I need?

At launch, Otto works with the Claude CLI (the Claude subscription many developers already have). Adapters for ChatGPT/Codex CLI, Gemini CLI, and direct API keys (OpenAI, Anthropic, OpenRouter) follow shortly after.

Does my voice or data get uploaded?

No. Speech-to-text and text-to-speech run on your machine using open-source models. Only the text of your request goes to the AI provider you connected — the same as when you use it directly.

Can it delete files or send emails without asking?

No. Otto works on graduated autonomy: you grant permissions per skill, destructive actions always require confirmation, and a session kill switch stops everything instantly.

Which platforms?

macOS first. Windows beta follows after launch. Mobile companion later.
